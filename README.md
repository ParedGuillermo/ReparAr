# ReparAr

Landing web de servicios técnicos en Corrientes Capital.

## ¿Qué ofrece este sitio?

- Reparación y mantenimiento de consolas PlayStation.
- Soporte técnico para estudiantes de sistemas.
- Acceso directo por WhatsApp para consultas rápidas.

## Tecnologías usadas

- React
- Vite
- Tailwind CSS
- ESLint

## Requisitos previos

- Node.js 18 o superior
- npm

## Instalación

```bash
npm install
